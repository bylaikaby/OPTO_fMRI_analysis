GETEXPS - Get EXPS from group(s).
EXPS = GETEXPS(SESSION, GrpName)
  EXAMPLE : exps = getexps(SESSION,{'movie1','movie2'})
                      exps = getexps(SESSION,'movie1')
  SEEALSO : getexpfromgrp, validexps,
                      any project files like xmovphys,xmovmri.
