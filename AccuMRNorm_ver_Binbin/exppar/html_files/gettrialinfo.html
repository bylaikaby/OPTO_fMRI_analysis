GETTRIALINFO - Returns information regarding individual trials
  GETTRIALINFO (SesName,GrpName/ExpNo) returns information regarding the
  trials within an observation period for SesName and group
  "GrpName" or "ExpNo".

  EXAMPLE :
      gettrialinfo('c01ph1',17)
      par = getsortpars('c01ph1',17)
      [IDX,PAR] = findtrialpar(par,0);
  VERSION : 0.90 18.04.04 YM    first release

  See also GETSORTPARS, FINDTRIALPAR, GETSTIMINFO
