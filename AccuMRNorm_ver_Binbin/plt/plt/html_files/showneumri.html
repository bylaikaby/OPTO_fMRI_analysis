SHOWNEUMRI - Displays the Neural Signals superimposed on the BOLD response.
  SHOWNEUMRI (SesName, ExpNo, RoiName, Thr) makes a figure with two subplots. The
  left shows the correlation map, and the right the time courses of the neural and MRI
  signals.

  USAGE:
      SHOWNEUMRI (SesName, ExpORgrp, RoiName, Thr)
              SesName        : Name of the session (e.g. k005x1)
              ExpORgrp      : Experiment number of Group Name; if negative uses RAW data and applies
                                          user-defined preprocessing (definition in description file under the
                                          ANAP.mareats field.
              RoiName        : ROI name (e.g. V1, V2,...)
              Thr                : Threshold for r values
    
      SHOWNEUMRI ('k005x1', 'p2c100');
      SHOWNEUMRI ('k005x1', -3) -- use tcImg of exp 3, preprocess, plot...
    
      The following function call shows good undershoot for brief pulses
      SHOWBPULSE ('f01pr1','polarflash');
