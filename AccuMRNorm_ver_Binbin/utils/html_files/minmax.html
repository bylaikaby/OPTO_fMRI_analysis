MINMAX - Ranges of matrix rows.
pr=MINMAX(p)
Syntax
  pr = range(P)
Description
  MINMAX(P) takes one argument,
      PR - RxQ matrix.
  and returns the Rx2 matrix PR of minimum and maximum values
  for each row of M.
Examples
  p = [0 1 2; -1 -2 -0.5]
  pr = minmax(p)
