ESS2SMR - exports adf/adfw/Cln data as 'smr' format for CED's Spikes2
    ESS2SMR(ADFFILE) exports adf/adfw data as 'smr' format.
    ESS2SMR(SESSION,[GRPEXP]) exports Cln dta as 'smr' format.

    EXAMPLE :
        >> ess2smr('y:/temp/testsmr.adfw')
